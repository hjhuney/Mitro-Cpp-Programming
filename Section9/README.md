## If Statement

Examples

```
// example 1
if (num > 10)
  ++num;
  
// example 2
if (health < 100 && player_healed)
  health = 100;
  
// example 3
if (selection == 'A')
  cout << "You selected A";
```

To execute more than 1 statement if condition is true, we use block statement

Block Statement Examples

```
if (num > 10) {
  ++num;
  cout << "that's numberwang!";
}
```
